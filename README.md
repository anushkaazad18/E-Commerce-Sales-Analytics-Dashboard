# E-Commerce Sales Analytics Dashboard

## Overview

This Tableau dashboard analyzes retail sales data to uncover insights related to revenue, profitability, customer behavior, product categories, and regional performance.

## Tools Used

* Tableau Public
* Data Visualization
* Business Analytics
* CSV Dataset

## Dashboard Features

* KPI Tracking
* Monthly Sales Trend Analysis
* Regional Performance Analysis
* Category Performance Analysis
* Top Customer Analysis
* Geographic Sales Analysis

## Tableau Public Dashboard

https://public.tableau.com/app/profile/anushka.sodha/viz/E-CommerceSalesAnalyticsDashboard_17822357118370/Dashboard2?publish=yes

## Key Insights

* West region generated the highest sales revenue.
* Technology category delivered the highest profit.
* Furniture showed lower profitability despite strong sales.
* Top customers contributed significantly to overall revenue.
